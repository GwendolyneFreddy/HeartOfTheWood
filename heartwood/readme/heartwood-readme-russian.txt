Сердце Леса (для BG2 - Трон Баала)

Описание: Мод добавляет в игру мощный магический посох, с помощью которого друид может вызывать себе на подмогу сильных лесных созданий и монстров. Посох находится во владении Кайланда Линда (Роща Друидов). Чтобы получить посох, вам, разумеется, придется сперва убить его хозяина.

Обращаем ваше внимание на то, что Кайланд Линд при попытке отнять у него посох, без раздумий применит его против вас и вашей группы. Будьте предельно осторожны.

Важно: Только друиды могут использовать этот посох.

А теперь - наслаждайтесь, этот мод сделан для вас!
